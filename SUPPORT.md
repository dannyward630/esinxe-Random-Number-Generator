# Support

Use [GitHub Discussions](https://github.com/dannyward630/esinxe-Random-Number-Generator/discussions)
for design questions and usage help when Discussions are enabled. Use
[GitHub Issues](https://github.com/dannyward630/esinxe-Random-Number-Generator/issues)
for reproducible bugs and concrete feature requests.

Before opening an issue:

1. Check the [README](README.md), [API reference](docs/API.md), and
   [algorithm specification](SPEC_V1.md).
2. Confirm the behavior against the latest `main` branch.
3. Reduce the report to a seed, key, language, expected value, and actual value.

Security-sensitive reports must follow [SECURITY.md](SECURITY.md).
